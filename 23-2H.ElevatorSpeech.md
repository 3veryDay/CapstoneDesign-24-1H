ViT는 매우 좋은 성능을 보이는 비전 모델이지만, 대규모의 이미지 데이터를 처리할 때 연산 비용이 상당히 크다는 단점이 있습니다. 이로 인해 모델의 효율성과 속도가 떨어질 수 있습니다. 따라서 저희는 이러한 문제를 해결하고자 합니다. 저희는 Spatial Locality를 활용한 Token Merging을 사용합니다. 이 기술은 이미지의 locality를 고려하여 인접한 토큰을 묶어 하나의 토큰으로 병합하는 방법입니다. 이를 통해 모델의 입력 토큰 수를 줄이고, 불필요한 연산을 최소화하여 모델의 가속화를 진행합니다. 이 방법을 사용하면 모델의 inference 속도가 빨라져서 실제 실시간 컴퓨터 비전 어플리케이션에 적합한 모델을 만들 수 있습니다. 또한, 자원 효율성이 높아집니다. 병렬 처리나 연산량 감소로 인해 필요한 하드웨어 리소스가 줄어 사용자들이 모바일 디바이스에서도 더 경제적으로 모델을 실행할 수 있다는 장점으로 이어집니다. 그리고 저희는 현재까지 아이디어 구체화와 코드 작성을 마쳤고, 실험을 통해 결과를 분석하고 문제점을 파악하여 코드를 수정할 예정입니다. 이상으로 스피치를 마치겠습니다. 감사합니다!
